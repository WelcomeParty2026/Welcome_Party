\subsection{備考}
\begin{itemize}
    \item 利用始めにドライヤーの有無と忘れ物がないか確認する
    \item 鍵の管理は厳重に行い，貸し借りの際はSlackで報告
    \item スタッフも原則21時までにお風呂に入っておく
    \item 準備が終わり次第，次の工程へ移行
    \item 入浴がスムーズに行えるように、誘導スタッフはタイミングを見計らって指示を出す
    \item 飲酒した場合は入浴できない
\end{itemize}
