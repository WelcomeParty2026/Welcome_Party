\subsection{備考}
\begin{itemize}
  \item 朝のつどいまでにベッド，部屋の片付けが完了しなければ朝食終了後に行う
  \item スタッフは布団のたたみ方を指導できるようにしておく
  \item 体調不良者の対応を行う
  \item 教職員の棟の起床係は先生方を軽く起こして朝食の有無を聞く
  \item 朝食がいらないと言われた場合，08:15には起床していただきたいことを伝える
  \item 手の空いているスタッフは宴のゴミの片付けを手伝う
  \item 起床係は7:30には荷物をプレイルームに置いて朝食を食べに行くことを指示する
  \item 先生方が朝食を食べられる場合は，教職員の棟の起床係が誘導を行う

\end{itemize}
