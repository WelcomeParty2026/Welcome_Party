\subsection{備考}
\begin{itemize}
  \item 翌日に車を運転するあるいは早朝に仕事が割り振られているスタッフは．作業に支障をきたさないようにする
  \item 飲み過ぎにより，先生方・スタッフ・新入生に迷惑をかけない(特にスタッフは飲み過ぎて翌日のイベントに支障をきたす恐れがあるため飲み過ぎ厳禁)
  \item 棟内は禁煙であるため，喫煙は所定の場所へ移動する(喫煙スタッフ適宜お供に付く)
  \item ゴミを分別する
  \item 酒類のゴミ(缶や注がれた紙コップ)は，一度水洗いしてゴミ袋に入れる
  \item スタッフは率先して先生方にお酒を注ぐ
  \item 施設全体として原則禁酒であるため，お酒をミーティング室から持ち出さない
  \item 新入生をミーティング室へ侵入させない
  \item 体調の悪い人は部屋で待機する
  \item 飲酒した場合は入浴できないことを先生方に伝え，入りたい人は先に入浴していただく
  \item 20歳未満の人は侵入禁止
\end{itemize}
